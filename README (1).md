# yorozu_car(車屋さん)
​
## サイト概要

### サイトテーマ
車の部品や車の修理・販売を目的としています。

### テーマを選んだ理由
私の身内が車の整備士をしており、いつか独立して車屋さんになりたいと言っていたので。
​
### ターゲットユーザ
老若男女問わず。
​
### 主な利用シーン
車の修理が必要な方、車の購入を検討中の方に使用して頂きたいです。
